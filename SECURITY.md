# Security Policy

## Supported Versions

We actively provide security updates and patches for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

As a secure vault application, security is our absolute highest priority. If you identify any vulnerability, leak, or potential cryptographic weakness, please report it immediately:

1. **Email us**: Send a detailed description of the vulnerability to the maintainers at shohanurrahman2580@gmail.com.
2. **Include Proof of Concept (PoC)**: Provide step-by-step instructions to reproduce the issue.
3. **Do NOT open public issues**: Please allow us to patch the issue and coordinate release schedules before disclosing details publicly.

We aim to acknowledge and triage reports within 24-48 hours.
